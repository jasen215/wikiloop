# WikiLoop 国际化（i18n）实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为 WikiLoop 实现中/英双语支持，Web UI 前端动态切换，系统托盘菜单根据 config.yaml 中的语言设置显示对应文字，首次启动时自动检测系统语言并写入 config.yaml。

**Architecture:** config.yaml 中 `ui.language` 为单一数据源（zh/en）；Web UI 每页内嵌 i18n 字典，`DOMContentLoaded` 时 fetch `/api/settings` 读取语言并替换所有 `data-i18n` 标记元素；托盘在 `Run()` 启动时接收 lang 参数，用对应文字初始化菜单；语言切换后 Web UI 立即刷新生效，托盘重启后生效。

**Tech Stack:** Go（config/tray/api 层），原生 JS（前端 i18n），无新依赖。

## Global Constraints

- 语言仅支持 `"zh"` 和 `"en"` 两个值，无其他语言
- `config.yaml` 是语言设置的唯一数据源，CLI 和 MCP 工具描述保持英文不变
- 首次启动（`ui.language` 为空）时检测系统语言并回写 config.yaml
- Web UI 语言切换立即生效；托盘语言切换重启后生效，settings 页需提示用户
- 所有 HTML 元素保留英文原文作为 fallback（JS 加载失败时降级）
- Go 文件遵循 `//go:build fts5` / `//go:build darwin` 等已有 build tag 规范

```go
func detectSystemLanguage() string {
    for _, env := range []string{"LANG", "LC_ALL", "LANGUAGE"} {
        if v := os.Getenv(env); strings.HasPrefix(v, "zh") {
            return "zh"
        }
    }
    if runtime.GOOS == "darwin" {
        out, err := exec.Command("defaults", "read", "NSGlobalDomain", "AppleLocale").Output()
        if err == nil && strings.HasPrefix(strings.TrimSpace(string(out)), "zh") {
            return "zh"
        }
    }
    return "en"
}
```

```html
    if (dict[key]) el.textContent = dict[key];

    const key = el.getAttribute("data-i18n-placeholder");
```
